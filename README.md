# Anomaly Detection in Electricity & Gas Consumption – Sonalgaz Data

An academic project supervised by **Not Yet**.  
The goal is to construct meaningful features from raw consumption data (electricity and gas) provided by **Sonalgaz**, then apply unsupervised machine learning models (Isolation Forest, HDBSCAN) to identify outliers, consumption imbalances, and unusual patterns in order to enhance the company in field agents inspections.

## 🎯 Project Objectives

- Extract and merge raw Excel files from Sonalgaz (electricity + gas invoices). (finished waiting for supervisor approval)
- Engineer features that capture energy differences, ratios, and normalized balances. (finished waiting for supervisor approval)
- Visualize Differenet Features with different approachs for better data understanding. (not yet)
- Train **Isolation Forest** to detect anomalous consumption records. (not yet/may not be)
- Apply **HDBSCAN** clustering to discover natural groupings and potential fraud/errors. (not yet /may not be)
- Validate results under the supervision of **our professor**.
- Test it in field by detecting innormal consumption and inspect it.
